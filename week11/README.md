# Week X：Docker 部署 ROS2 环境实验
## 实验内容
本周完成了以下任务：
1.  安装并配置 Docker 环境
2.  构建 ROS2 基础镜像 `my-ros2-full:v1.0`
3.  启动 ROS2 容器并验证端口映射
4.  查看运行中的容器状态

## 运行命令
```bash
# 查看正在运行的 Docker 容器
docker ps

# （补充：启动 ROS2 容器的典型命令，含端口映射）
docker run -d -p 6080:80/tcp --name ros2-container my-ros2-full:v1.0
```

---

## 关键信息解读
终端 `docker ps` 输出的关键信息：
- **容器ID**：`b5cf969c73bd`
- **镜像名称**：`my-ros2-full:v1.0`，为自定义构建的 ROS2 完整环境镜像
- **端口映射**：`0.0.0.0:6080->80/tcp`，表示宿主机的6080端口映射到容器的80端口，可通过浏览器访问容器内的服务
- **运行状态**：`Up About a minute`，容器已正常启动并运行约1分钟

---

## 遇到的问题
1.  **问题**：运行 `docker ps` 看不到预期的 ROS2 容器
    **解决**：确认容器是否以 `docker run -d` 方式后台启动，若未启动需重新执行容器启动命令；若容器已退出，使用 `docker logs <容器ID>` 查看退出日志排查错误。

2.  **问题**：无法通过 `localhost:6080` 访问容器内的服务
    **解决**：检查宿主机防火墙是否放行6080端口，确认容器内服务是否在80端口正常监听，核对启动命令的端口映射配置是否正确。
